# kubernetes-labs

Projetos praticos de Kubernetes do basico ao intermediario.

## Estrutura

| Modulo | Conceito |
|---|---|
| 01-pod-basico | Menor unidade do Kubernetes |
| 02-deployment | Gerenciamento de replicas |
| 03-service | Exposicao de aplicacoes |
| 04-configmap-secret | Configuracoes e dados sensiveis |
| 05-namespace | Isolamento de ambientes |

## Pre-requisitos

- kubectl instalado
- Cluster Kubernetes rodando (Docker Desktop, Minikube ou cloud)
